## Class Registration

### How to Sign Up
1. Check availability on member portal calendar
2. Register online or via signup sheet on bulletin board
3. Payment due at registration (online or cash/check)
4. Confirmation email sent 24 hours before class

### Cancellation Policy
- Full refund if cancelled 48+ hours before class
- 50% refund if cancelled 24-48 hours before
- No refund for cancellations under 24 hours
- Classes may be cancelled if minimum enrollment not met

### Waitlist
Popular classes fill quickly. Add yourself to the waitlist - we often add second sessions if demand is high.

## Private Events

The workshop can be reserved for private events:
- **Members**: $50/hour (4 hour minimum)
- **Non-members**: $100/hour (4 hour minimum)
- Includes facility access, basic hand tools
- Must provide certificate of insurance for large events
- Contact shop manager for availability

## Tool Maintenance Schedule

Some tools are periodically unavailable for maintenance:

- **1st Tuesday morning**: Table saw blade changes and alignment
- **2nd Monday**: CNC router calibration and maintenance
- **3rd Monday**: Dust collection system servicing
- **4th Monday**: Metal shop equipment maintenance

Check the member portal for real-time tool availability.

## Getting on the Calendar

Want to teach a class or organize an event?
1. Submit proposal to shop manager
2. Include: topic, duration, max participants, materials needed
3. Board reviews proposals monthly
4. Popular classes may become recurring events!
