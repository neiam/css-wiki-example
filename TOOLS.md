# Tools

An overview of the tools available in the cooperative workspace.

## Workshop Tools

### [Saw](TOOLS/SAW.md)
For cutting wood and other materials.

### [Hammer](TOOLS/HAMMER.md)
For driving nails, general assembly, and demolition work.

## Tool Policies

### Borrowing Tools
- Sign out tools using the checkout sheet
- Return tools to their designated locations
- Clean tools after use

### Maintenance
- Report damaged or dull tools
- Participate in monthly tool maintenance days
